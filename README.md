## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo.

```sh
git clone https://github.com/leodaiub/floor-plan.git
```

2. Install the NPM dependencies.

```sh
npm install
```

3. Start the development server.

```sh
npm run dev
```
